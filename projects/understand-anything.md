# Understand-Anything

## 一句话定位
将任意代码转换为交互式知识图谱，支持探索、搜索和问答，兼容 Claude Code、Codex、Cursor、Copilot、Gemini CLI 等主流 Coding Agent。

## 它解决的问题
Coding Agent 理解大型代码库时 token 消耗巨大且理解不深。Understand-Anything 通过将代码转为交互式知识图谱，让 Agent 和开发者都能更高效地理解代码。

## 为什么值得关注
本周 +23K stars，GitHub Trending 周度 Top 1。与 CodeGraph 共同定义了"Agent 代码理解工具链"赛道。动态交互式图谱区别于静态索引，提供可探索、可搜索、可问答的深度理解能力。
