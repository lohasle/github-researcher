## 趋势信号总结

### Multi-Agent Orchestration 爆发（持续）
- oh-my-claudecode (7832/week)
- oh-my-codex (14247/week)
- hermes-agent (9662/week)

**判断：** 多 Agent 协作从概念走向工程化，以"团队"为原子的编排方式正在形成新范式。

### Voice AI 基础设施化（持续）
- VibeVoice (ASR 进 HuggingFace Transformers)
- Realtime TTS (0.5B, streaming)

**判断：** Voice AI 正在从"能力展示"转向"基础设施组件"。

### Claude Code 生态扩张（持续）
- oh-my-claudecode
- oh-my-codex
- claude-howto (模板/指南，12085/week)

**判断：** Claude Code 正在成为 AI Coding 平台，其生态周边价值显现。

### MCP 协议扩散（新趋势）
- GitNexus（MCP Server for codebase）
- qmd（MCP Server for local docs）
- hermes-agent（MCP Gateway）

**判断：** MCP 正在从"AI Agent 协议"演化为"AI Native 应用协议层"，协议层基础设施地位确立。

### 端侧 LLM 推理生产化（新信号）
- LiteRT-LM（Google，已在 Chrome/Pixel Watch 生产落地）
- Gemma 4 端侧部署

**判断：** 端侧 LLM 推理不再是 Demo，开始进入真实产品。值得持续关注端侧推理框架赛道。
