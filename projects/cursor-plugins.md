# Cursor Plugins

## 一句话定位
Cursor 官方插件规范和官方插件集合，定义了 Cursor 平台的插件标准。

## 它解决的问题
Cursor 作为主流 Coding Agent 之一，缺乏统一的插件标准。本仓库提供了插件规范和官方示例。
