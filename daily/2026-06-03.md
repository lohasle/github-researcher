### 2. fff — AI Agent 文件搜索

**是什么：** Rust 实现的高速文件搜索工具包，支持 MCP/Nvim/Pi，提供 Frecency 排序、智能大小写、定义行优先等 Agent 友好特性。

**为什么火：** Agent 每次搜索都在浪费 Token 做 grep。fff 提供结构化、排序后的结果，直接减少 Token 消耗。

**定位判断：工具型，但可能成为 Agent 工具链标准组件。**
