# diffusionstudio/lottie

## 一句话定位
AI 生成生产级 Lottie 动画：通过 Claude Code 或 Codex 直接生成可用的 Lottie 动画文件。

## 它解决的问题
Lottie 动画是移动端和 Web 的工业标准格式，但制作 Lottie 需要专业设计工具和技能。该项目让 AI 直接生成生产级 Lottie 动画。

## 为什么值得关注（2026-06-11）

1. **2K⭐ 周增速** — 本周最高增速之一
2. **AI → 生产资产** — 不是概念验证，直接生成可用的 Lottie 文件
3. **Agent 集成** — Claude Code / Codex 原生支持
4. **设计工具链 AI 化** — 代表设计工具链 AI 化的趋势

## 热度来源判断
- Lottie 是刚需格式 + AI 生成设计资产正在爆发 + Agent 工具链集成

## 关键技术亮点
- Claude Code / Codex 集成，Agent 直接调用
- 生成标准 Lottie JSON 格式，兼容所有 Lottie 播放器
- TypeScript 实现

## 架构启发
**启发：** AI 生成设计资产的关键不是「能生成」，而是「生成的东西能直接进产品」。Lottie 格式的标准化使得 AI 输出可以直接进入产品流程。

## 定位判断
**工具型。** 有明确的单一用途，不会演化为平台。

## 风险/局限/泡沫点

3. **依赖模型能力** — 动画生成质量高度依赖 LLM 理解设计意图的能力
