# ViMax

## 一句话定位
Agentic 视频生成系统 — 导演、编剧、制片人和视频生成器 All-in-One，通过多 Agent 协作完成视频制作全流程。

## 它解决的问题
视频生成缺乏系统化的生产流程管理。ViMax 通过将视频制作分解为多个 Agent 角色（导演、编剧、制片人、生成器），实现了 Agentic 的视频生产流程。

## 为什么值得关注
HKUDS 出品，Agentic 多模态的代表项目。7.9K stars 表明市场对"AI 导演"概念有认可。
