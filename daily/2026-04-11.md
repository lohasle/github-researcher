## 重点分析

### superpowers - AI 编程 Agent 的方法论革命

**定位**：AI 编程 Agent 技能框架，强制执行结构化开发流程

**为何火爆**：
- 解决 AI 编程 Agent "直接写代码"的通病——缺乏设计思考
- 强制执行 brainstorm → spec → TDD → review 工作流
- 被 Anthropic Marketplace 官方收录，生态认可度高
- 3 个月内从 0 到 121K stars，增速惊人

**技术亮点**：
- 技能组合化设计：每个技能独立可复用，支持跨 Agent 使用

- 支持多种 Agent 平台：Claude Code、Cursor、Codex 等
- 内置代码审查循环：确保代码质量

**架构启发**：
- 行为优先而非工具优先的 Agent 设计
- 技能即约束的开发哲学
- 从"让 AI 写代码"到"让 AI 遵循工程实践"的范式转变

**技术亮点**：

## 今日行业动态

### GitHub Copilot Workspace 添加多仓库上下文
GitHub 宣布 Copilot Workspace 支持跨仓库上下文感知，开发者可以在微服务和 monorepo 架构间无缝工作。这意味着 AI 编程工具从单仓库视角扩展到全局架构视角。

### MCP 支持全面铺开
GitHub Copilot 正式支持 MCP（Model Context Protocol），开发者可以通过 MCP Server 交互 issues、PR 等功能。MCP 正在从"AI Agent 协议"演化为"AI Native 应用协议层"。
